# LLM-Based Text Classification

## Overview
Built a text classification pipeline using Large Language Model preprocessing techniques including tokenization and dataset augmentation.

## Features
- Data preprocessing using Pandas
- Tokenization pipeline
- Model-ready dataset preparation
- Scalable pipeline for NLP tasks

## Tech Stack
- Python
- Pandas
- NLP Tokenizers

## Use Case
Can be used for:
- Fake news detection
- Sentiment analysis
- Text categorization
